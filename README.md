# Gandalf

sql client built with electron

## Installation Instructions:

-   make sure you have node and npm installed
-   clone this repo
-   cd into the clone
-   run

```console
npm install
npm start
```

## Develop

```console
# To compile and watch typescript
npm run build-watch

# Run In another tap or window
npm run dev

# or with dev tools open
npm run dev-with-tools
```
